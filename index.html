Rust meets This War of Mine Project

A side-scrolling game inspired by "This War of Mine" game from 11bit Studios mixing the rough emergent free gameplay of Rust from Facepunch Studios.

The following systems and mechanics are already being prototyped and tested:
Base building, crafting, inventory, equipment (tools, weapons), natural resources (ore nodes, trees, stone nodes), storage.

Shader Tool

Rendering application developed in C++ and DirectX12 as final project for my MSc in Computer Science for Games Development at the University of Hull in the UK. It consists of a node-graph viewport where nodes can be added and connected through pins to generate the visual effect from the imported shader.

AAA Arcade Racing Game for a Major Publisher in UE5

Worked for 2 years on a project of an (yet) unannounced AAA arcade racing game in Unreal Engine 5 at Stellar Entertainment in Guildford - UK.

My contribuition was mostly on the gameplay code and building the first version of the multiplayer lobby using Epic Online Services (EOS).
